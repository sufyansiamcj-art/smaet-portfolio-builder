# 🚀 Smart Portfolio Builder

**Smart Portfolio Builder** هو تطبيق ويب ذكي مبني باستخدام **Python (Flask)** و **Tailwind CSS**، يهدف إلى تمكين المبرمجين من إنشاء محفظة أعمال (Portfolio) احترافية في ثوانٍ معدودة عن طريق سحب بيانات مشاريعهم مباشرة من **GitHub API**.

!

## ✨ المميزات (Features)
- **جلب تلقائي:** سحب المستودعات (Repositories) العامة، الأوصاف، لغات البرمجية، وعدد النجوم.
- **تصميم عصري:** واجهة مستخدم مظلمة (Dark Mode) متجاوبة بالكامل باستخدام Tailwind CSS.
- **نظام التصدير:** إمكانية تحميل المحفظة كملف HTML مستقل وجاهز للنشر.
- **أداء سريع:** كود نظيف وخفيف يعتمد على Flask في الخلفية.

## 🛠 التكنولوجيا المستخدمة (Tech Stack)
- **Backend:** Python, Flask
- **Frontend:** HTML5, Tailwind CSS, Jinja2
- **Integrations:** GitHub REST API
- **Deployment Ready:** Gunicorn & Render/Railway configuration
